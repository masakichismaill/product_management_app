# 商品管理アプリ

Flask と SQLite を使って作成した、シンプルな商品管理アプリです。  
商品の登録・編集・削除に加えて、在庫管理や売上履歴の確認ができます。

## 概要

このアプリは、商品情報を管理しながら、販売や補充に応じて在庫数を更新できるようにしたWebアプリです。  
売上履歴の記録や、商品ごとの売上回数・日別売上の集計も確認できます。

## 主な機能

- 商品の追加
- 商品情報の編集
- 商品の削除
- 在庫の販売処理
- 在庫の補充処理
- 売上履歴の表示
- 商品ごとの売上回数の集計
- 日別売上の集計

## 使用技術

- Python
- Flask
- SQLite
- HTML
- CSS

## 工夫した点

- 商品の登録・編集・削除だけでなく、販売と補充を分けて管理できるようにしました。
- 売上履歴を残せるようにし、あとから販売状況を確認できるようにしました。
- 商品ごとの売上回数や日別売上を表示し、簡単な集計ができるようにしました。
- シンプルな構成にして、基本的な商品管理の流れが分かりやすいようにしました。

## 想定している用途

- 小規模な商品管理
- 在庫数の確認
- 販売履歴の記録
- 売上状況の簡単な集計
